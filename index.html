Why haven?

Haven is a free and open-source meditation timer. Its simple user interface makes it very easy to use, so you can focus on meditation without getting distracted.

Key Features

  • Clean and uncluttered interface.
  • Use it offline like a native app.
  • Calming dark mode for night-time use.
  • User-friendly and accessible for everyone.
  • Haven's code is openly visible for everyone.

How to Use

  1. Install Haven by adding it to your home screen from your browser.
  2. Choose your meditation time and set the timer.
  3. Begin meditating and enjoy the calm.
